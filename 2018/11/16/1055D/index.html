Refactoring

题目

Alice写了个程序，想使用IDE的一键重构功能重构变量名，重构的方式如下：
确定源字符串$s$和目标字符串$t$，然后把所有变量中第一个出现的$s$子串替换成$t$。
要你找到这样的$s$和$t$或者告诉她这不可能。

输入

第一行包含一个整数$N (1 \leq N \leq 3000)$。
接下来$N$行，每行都是需要重构的变量名，$w_1, w_2, \ldots, w_n$。
接下来$N$行是需要重构的结果，${w^{‘}}_1, {w^{‘}}_2, \ldots, {w^{‘}}_n$。
保证每个变量重构前后的长度一样。
至少有一个变量重构前后不相等。
每个变量都只包含小写字母。

输出

如果不能执行重构，输出NO。
否则先输出YES，然后两行分别输出$s$和$t$。

分析

找到每个变量必须改变的部分，这个部分代表每个需要改变的变量都必须一样。
然后向左右尽量延伸，最后判断替换后的每个变量是否等于目标变量。
